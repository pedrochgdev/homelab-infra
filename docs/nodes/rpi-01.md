# rpi-01

## Summary

`rpi-01` is the first Raspberry Pi node in the homelab and is intended as an active lab platform for experimentation, lightweight services, and future cluster-oriented work.

It is already part of the environment, even though it does not currently run a defined workload.

## Role

- active lab node
- future cluster candidate
- platform for experimentation and learning

## Operating System

- Platform: Ubuntu Server
- Version: `24.04`

## Hardware

- Model: Raspberry Pi 5
- Memory: 16 GB RAM
- Storage: 512 GB NVMe

## Current State

`rpi-01` is installed and present in the rack environment, but it does not yet host any active services or assigned workloads.

It should not be treated only as a placeholder for future clustering. It is available now as a practical experimentation node.

## Physical Notes

- mounted in the rack environment
- not currently in its intended rackmount due to fit or compatibility issues

## Intended Direction

The long-term direction for `rpi-01` includes:

- cluster experimentation
- Kubernetes-related learning
- orchestration testing

However, a broader cluster design has not yet been finalized, and additional nodes have not yet been added.

## Near-Term Use Cases

Before becoming part of a larger cluster, `rpi-01` can be used for:

- Linux administration practice
- container experimentation
- lightweight internal services
- testing and learning workloads
- early orchestration research

## Documentation Notes

This node should remain documented as an active part of the lab even before its long-term role is finalized.
