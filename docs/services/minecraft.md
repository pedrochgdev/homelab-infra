# Minecraft

## Summary

A Minecraft server is planned as a future workload within the homelab. It is currently intended to run as a dedicated virtual machine on `virt`.

This service is not yet deployed.

## Planned Host

- Physical host: `virt`
- Deployment model: virtual machine
- Status: planned

## Purpose

The planned purpose of this service is:

- provide a self-hosted Minecraft server
- expand the homelab with another practical service workload
- gain additional experience with VM-based service hosting

## Current Design Direction

The Minecraft server is expected to run in its own VM rather than being combined with existing application or infrastructure services.

This is consistent with the broader homelab design principle of keeping workloads separated by role.

## Notes

Resource allocation, operating system details, access method, and game configuration are still to be defined once deployment begins.
